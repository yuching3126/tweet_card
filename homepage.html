Overview

Overview

Tweet C.A.R.D. API，這個名字取推薦卡片—推卡的諧音，而C.A.R.D.為 credit card analysis and recommendation的縮寫。

有鑑於市面上已有的優惠查詢方式較為被動並且無法做個人化之推薦，希望Tweet C.A.R.D. API 能夠結合微時刻與開放銀行。

微時刻是由Google於2015年提出的概念，此概念包含四個重要關鍵的時刻。其中，消費者產生"購買意圖"的時刻是此API著眼的時刻。

本產品以API 的形式開發，可提供不同功能的APP串接，實用性高。

透過APP的使用，了解使用者的消費習慣找出最適當時機將訊息傳遞到適合的使用者。

而開放銀行由金管會分為三個階段，於2019年八月底開放第一階段「公開資料查詢」。

於是，透過銀行提供公開信用卡優惠內容，此API得以自動且即時更新信用卡優惠內容

除了向APP使用者提供推薦服務以外，此API將提供雙向的服務。

本產品將把後台數據分析後提供給銀行端，包括辦卡率、用卡率、此卡常用的消費地點及品牌、以及此卡在自家銀行及其他銀行中的相關排名等等。

開放銀行目前處於第一階段並邁向第二階段。本產品將隨著開放銀行不同階段逐步開放並成熟發展後，推出更多服務。
